Gameboy Project

Programmed games (Snake, Skyfall) in Arduino IDE with Raspberry Pi Pico, buttons, and ST7789 240x320 TFT Display. Currently in the process of designing a 3D printed case for the device, will be soldering components onto a perfboard with a Battery Management System IC, Class-D Amplifier, and Speaker to make the device more portable and enjoyable. Will also add MicroSD IC to allow for more complex games.

CPU Project

Designed a 32-bit pipelined RISC-V CPU in Logisim with ALU, register file, immediate generator, and control logic, with a 2-stage IF/EX pipeline supporting arithmetic, branches, loads/stores, and jumps, plus branch/jump flushing for control hazards.
